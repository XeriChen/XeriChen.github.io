ESTJ分析
#

基本信息
#

  • “name”: 外向 直觉 思考 判断
  • “nameDescription”: 外向型思维与内向型感觉
  • “epithet”: 守护者

“description”
#

作为一名 ESTJ，你的主要生活模式是对外的，你会理性、逻辑地处理事情。你的次要模式是内在模式，你通过五官以直观、具体的方式接受事物。ESTJ 生活在一个充满事实和具体需求的世界里。他们活在当下，眼睛不断扫视个人环境，以确保一切都在有条不紊地顺利进行。他们尊重传统和法律，有一套明确的标准和信仰。他们对他人也有同样的要求，对不重视这些制度的人没有耐心，也不理解他们。他们重视能力和效率，喜欢看到自己的努力能迅速见效。ESTJ 是负责任的人。他们对事物的发展方向有着清晰的认识，因此会自然而然地进入领导角色。他们自信且积极进取。他们在制定系统和行动计划方面极具天赋，能够洞察完成特定任务需要采取哪些步骤。他们有时会非常苛刻和挑剔，因为他们有如此坚定的信念，如果他们认为某人没有达到他们的标准，他们很可能会毫无保留地表达自己的观点。但至少他们的表达可以被当作表面现象，因为 ESTJ 非常直率和诚实。ESTJ 通常是模范公民和社区支柱。他（她）认真履行自己的承诺，严格遵守自己的 “好公民 “标准。ESTJ 喜欢与人交往，喜欢找乐子。

在社交活动中，尤其是以家庭、社区或工作为主题的活动中，ESTJ 可以表现得非常喧闹和有趣。ESTJ 需要注意过于刻板和过于注重细节的倾向。由于他们非常重视自己的信念，因此一定要记住重视他人的意见和看法。如果他们忽视了自己的感受力，就可能会在满足他人的亲密需求方面出现问题，并且可能会在不知不觉中伤害他人的感情，因为他们会在需要更多情感敏感度的情况下运用逻辑和理性。当被压力所困时，ESTJ 经常会感到与他人隔绝。他们觉得自己被误解了，价值被低估了，自己的努力被认为是理所当然的。虽然 ESTJ 通常非常善于言辞，在表达自己的想法方面没有任何问题，但当他们处于压力之下时，就很难将自己的感受用语言表达出来，也很难与他人沟通。ESTJ 把安全感和社会秩序看得比什么都重要，他们认为自己有责任尽一切可能来加强和促进这些目标的实现。他们会修剪草坪、参加投票、加入家长教师协会、参加业主协会会议，总之，他们会竭尽所能促进个人和社会安全。

ESTJ 几乎做任何事情都会付出很大的努力。在工作、婚姻和社区中，他们会以充沛的精力去做每一件他们认为应该做的事情。他或她认真、务实、现实、可靠。虽然 ESTJ 会尽职尽责地为某项事业或目标而努力，但他们可能不会自然而然地看到或重视那些超出其实际范围的目标的重要性。不过，如果 ESTJ能够看到这些目标与实际问题的相关性，那么可以肯定的是，他们会尽一切努力去理解这些目标，并将其纳入自己对清晰度和安全感的追求中。”

“jungianFunctionalPreference”
#

  • “auxiliary”: “内向感知”
  • dominant: 外向型思维
  • inferior: 内向的感觉
  • tertiary: 外向直觉

generalTraits
#

  • 天生的领导者–他们喜欢掌权
  • 重视安全和传统
  • 忠诚
  • 勤奋可靠
  • 运动健康
  • 有一套明确的标准和信念，并据此生活
  • 对无能或低效没有耐心
  • 出色的组织能力
  • 喜欢创造秩序和结构
  • 非常全面
  • 将跟踪项目直至完成
  • 直率坦诚
  • 履行职责的动力

relationshipStrengths
#

  • 热情、乐观、友好
  • 稳定可靠，可以依靠他们为家人提供安全保障
  • 努力履行职责和义务
  • 负责任地打理家里的日常事务
  • 通常善于（尽管保守）理财
  • 个人不受冲突或批评的威胁
  • 有兴趣解决冲突，而不是置之不理
  • 认真履行承诺，寻求终身关系
  • 关系破裂后能够继续前进
  • 必要时能够执行纪律

relationshipWeaknesses
#

  • 倾向于认为自己总是正确的
  • 总是需要负责的倾向
  • 对低效率和马虎不耐烦
  • 无法自然地与他人的感受保持一致
  • 天生不善于表达自己的感受和情绪
  • 可能无意中用不敏感的语言伤害他人
  • 注重物质和地位的倾向
  • 一般不适应变化和进入新领域

successDefinition
#

ESTJ 人格类型的人非常重视社会秩序和结构。在其一生中，ESTJ 型人格会形成一套判断标准，他们会用这些标准来对世界上存在的事件和印象进行排序。这些标准本质上是社会原则。ESTJ 非常坚信自己的原则，并坚决反对任何违反原则的行为。他们认为自己的原则定义了适当的行为和态度，因此应该无条件地遵守。就像他们天生会创造规则，因此是天生的领导者一样，ESTJ 也相信要遵守现有的社会规则。他们通常是领导者，但如果他们信任所遵循的制度的权威性，他们也很容易跟随。对于违反原则的行为，ESTJ 可能会相当严厉。对 ESTJ 来说，遵守原则比考虑违反原则的人的立场或感受更重要。在 ESTJ 将人际关系中的行为标准纳入其社会规则体系之前，他们的严厉态度可能会破坏人际关系。ESTJ 非常喜欢与他人相处，并希望促进传统的人际关系。如果 ESTJ 能够在自己确定的原则体系内生活，他们可能会觉得自己是成功的，但他们真正和持久的成功将来自于创造和维持良好而持久的原则的能力，从而能够充分和持续地处理生活中的各种情况。

strengths
#

  • 他们坚信要尽职尽责，并出于责任感而履行职责，而不是期望得到回报。
  • 他们通常记忆力很好。
  • 他们是天生的领导者。
  • 他们通常是优秀的战略家和 “游戏 “玩家。
  • 他们通常遵纪守法，勤奋工作。
  • 他们忠诚–对家人、朋友、国家等。

“gifts”
#

  • 他们理解并创造了无条件的有效而公平的原则，并以此为社会提供了巨大的服务。
  • 他们可能会成为杰出的法官和政治家。
  • 他们以高度的伦理道德方式生活。
  • 他们是伟大的战略家，可能成为伟大的军事指挥官。
  • 他们能够创建行为体系，以满足生活中遇到的所有需求。这样，他们就会感到满足，因为他们的世界是以原则为中心组织起来的，既有效又持久。

potentialProblemAreas
#

  • 可能不知道或不关心自己在他人眼中的形象。
  • 可能会故意欺负别人，让他们做出某种行为（理由是他们在执行一项原则）。
  • 可能会很快否定他人的意见，而没有真正加以考虑。
  • 可能难以向他人表达自己的想法和感受。
  • 可能难以理解考虑他人感受的重要性，也难以满足他们的情感需求。
  • 可能会记仇，难以原谅别人。
  • 可能脾气暴躁、急躁。
  • 可能对他人有很强的控制欲。
  • 可能无法确定个人生命的价值。
  • 可能无法看到其行为的长期影响。

explanationOfProblems
#

上述几乎所有有问题的特征都可归因于一个共同的问题，即外向思维超越了 ESTJ 人格，以至于所有其他人格功能都只是为了满足思维的需要而存在。一个健康和成功的人格需要在其主导功能和辅助功能之间取得良好的平衡。对于 ESTJ 来说，主导功能 “外向思维 “需要得到辅助功能 “内向感觉 “的支持。如果内向感官只是为了支持外向思维的议程而存在，那么这两种功能都没有发挥出自己的潜能，ESTJ 在工作或人际交往中也无法发挥出自己的潜能。在辅助感觉功能未得到充分利用的情况下，ESTJ 会完全在现有原则的范围内生活。他们会将自己的一套原则奉为不可剥夺的 “正确做法 “的代表，并将生活中遇到的一切都应用到这一原则体系中。如果他们发现有不符合自己原则的行为，他们就会无情地加以评判，并拒绝接受任何其他违反原则的观点。由于被外向型思维过程所束缚，他们失去了真正考虑传入信息的能力，因此也就失去了与他人协同合作、有效解决问题的能力。也许最重要的是，ESTJ 失去了与自我沟通的能力。他们变得与自己的个人需求脱节，与核心自我分离。这些情况的最终结果是，ESTJ 型领导者期望绝对服从自己的要求；他们缺乏洞察这些要求的长远影响的能力；他们不愿意考虑其他解决方案或计划；他们脱离了任何个人优先事项或价值体系。这样的领导者不可能在工作或个人生活中取得高效和成功，尽管他们很可能没有意识到自身问题的原因。让自己的主导功能凌驾于自己的个性之上是很常见的。就外向型思考者而言，如果让 “思考 “占据主导地位而不加以制衡，就会对 ESTJ 的社会交往产生重大影响。女性 ESTJ 可能会被视为盛气凌人、控制欲强或男性化，可能不那么容易被社会标准所接受。这可能会导致女性 ESTJ 缺乏自尊。男性 ESTJ 的情况更糟一些，因为社会的刻板印象可能会鼓励他们认为自己有权专横跋扈。他们可能没有兴趣超越自己有限的视野。外向思维是一种人格功能，它通过识别和遵守逻辑原则来建立结构。它是一种社会形式的判断，因为它是由外部世界而不是内心的是非观来定义的。外向思维的真正优势在于其高度的道德性。它不受个人诉求的左右，而是全心全意地相信原则的价值。如果原则存在，那么它就应该被遵守。一旦外向思维型的人确定了原则，他们的职责就是执行这一原则。外向型思考者不会因为遵循原则而期望得到任何回报。他们希望原则能得到无条件的尊重。换句话说，外向型思考者认为，你应该做正确的事，因为它是正确的，而不是因为你想得到回报才做正确的事。在这种理想意义上，外向思维判断力是法律和法律体系的基石。外向思维以其纯粹的意图，为我们的社会提供了很多东西。内向感知是 ESTJ 的辅助功能，是观察数据并将其储存起来以备将来参考的手段。内向感觉型能够考虑和存储大量数据。当这些数据被输入到主导功能 “思考 “中时，ESTJ 就会利用真实的数据来形成原则，并对世界实施结构化处理。如果 ESTJ 型人格的思考功能过于突出，那么辅助感官功能的重要性就会降低。数据不能脱离原则的背景来看待。ESTJ 无法看到违反原则之外的事实。他们无法客观地看待数据。

solutions
#

一個有興趣發掘自己潛能的 ESTJ，應該有意識地嘗試暫緩作出判斷，直至知道所有事實為止。有效的 ESTJ 不害怕在已知系统中无法理解或有效处理信息时重新定义原则。实践这种行为将有助于辅助性的内向型感觉的蓬勃发展，从而使整个人格成为一种更有效、更积极的力量。

例如，一位 ESTJ 朋友最近告诉我，他确信他母亲的清洁女工是个骗子。我问他为什么会这么想，他说：“因为她收了钱却什么都不做”。显然，房子的打扫没有达到他的标准。他认为，付给清洁女工钱是让她做某项工作，他希望她能按照一定的标准完成工作，并支付一定的费用。这是他的原则。她违反了这一原则，工作做得不够好。对他来说，她拿了钱却没有完成工作，这简直就是犯罪行为。当我问他是否有人指导过清洁女工具体打扫哪些地方时，他说他不知道，但她是一名清洁女工，所以她应该知道该怎么做（另一个原则）。当我进一步了解这一情况时（出于对朋友母亲的关心），我发现这位清洁女工几乎没有人指导她打扫什么，但她在受雇期间一直很忙，而且她可能不是最干净的清洁女工。当她得到更多的指导后，她的表现就更好了。我断言这位清洁女工在受雇期间一直很忙，但我的这位 ESTJ 朋友断然拒绝了我的说法。他不会考虑这一信息，也不会考虑她的工资远远低于清洁女工的标准工资这一事实。在他的原则体系中，她是一个骗子，他不会考虑可能提供另一种解释的数据。我的这位 ESTJ 朋友并没有简单地拒绝接受新的信息，而是稍稍改变了他的原则，以考虑到个人能力和个人成果的差异，并考虑到清洁女工的低工资确实在某种程度上抵消了较低的服务质量。利用新数据（由内向感知提供）来调整和重新定义他的原则，可以让他创建一个更有效的原则体系，从而更好地应对未来类似的挑战。总的来说，培养内向感知力是 ESTJ 优化其原则的有效性和应变能力的关键。这也会改善他们性格的总体平衡，从而为他们心理其他方面的成长打开大门。

suggestions
#

  • “愿意根据新的事实制定新的原则和改变现有的原则。”
  • “如果你生气了，就走开。当你允许愤怒控制你的行为时，你就输了，很可能别人也输了。在你处理完愤怒并冷静下来后，继续你的工作。”
  • “试着找出你在生活中经常遇到的每个人的性格类型。记住，直觉型的人有时会以间接、游离的方式说话。试着对此保持耐心，记住每个人都有自己的优点。”
  • “记住，你从那些与你截然不同的人身上学到的东西是最多的。无论你一开始多么难以发现，他们都能为你提供一些东西。”

livingHappilyTips
#

“有些 ESTJ 难以融入我们的社会。他们的问题通常与控制他人、对他人的行为抱有不合理的期望、不考虑他人的需求和想法以及通常表现得过于强势有关。这些问题主要源于 ESTJ 惯用的外向思维模式，即快速、自动地将现有原则应用于外部世界，而不是花时间根据自己的行为原则来权衡内心数据。ESTJ 需要能够利用丰富的内部数据为自己的原则提供依据。一味坚持坚定的原则，而忽视事实信息，就会带来麻烦，比如我们在历史上看到的各种独裁政权。为了健康地发展，ESTJ 必须认识到其内在数据存储的重要性，并需要利用所有可用的数据来形成良好的原则。为了实现这一目标，ESTJ 需要认识到内向的重要性，并开发利用其最高的内向功能–内向感应。 "

tenRulesToLive
#

  • “发挥你的优势！你被赋予了创造超越个人经验的逻辑和道德原则的伟大能力。在制定这些原则时，要考虑到所有可用的数据，让这些原则尽善尽美。”
  • “面对自己的弱点！看清自己的弱点，并努力克服它们。尤其要抵制过早下结论的倾向，牢记考虑他人感受的重要性。”
  • “了解事实或将其写下来。你需要通过事实来确定良好的生活原则。将它们口述或写在纸上对你来说可能是一个很有价值的工具。”
  • “接纳一切。不要因为自以为已经知道答案，就过早地否定各种想法。首先寻求理解，然后寻求被理解。”
  • “当你愤怒时，你就输了。你对自己原则的热情令人钦佩，但当你陷入 “愤怒陷阱 “时，这种热情就会变成破坏性的。请记住，愤怒会破坏人际关系，也会对他人造成极大的伤害。在你向他人发泄愤怒之前，先化解你的愤怒。只有以不针对个人和冷静的方式，才能有效地处理分歧和失望。”
  • “在人际关系中做你自己 不要期望自己是一个 “温柔体贴 “或 “温情脉脉 “的人。要认识到，你最热切的联系始于头脑，而非心灵。你希望你的行动能对你所爱的人说明一切。这对某些人来说可能还不够。注意他人的情感需求，用对你来说真实的语言表达你对他们真诚的爱和尊重。做你自己。”
  • “对自己负责。不要把生活中的问题归咎于他人。向内寻找解决方案。没有人比你更能掌控自己的生活。”
  • “谦虚。评判自己至少要像评判他人一样严厉。”
  • “抵制控制他人的冲动。你不能强迫他人遵守你的思维方式。你可能认为自己知道什么对别人最好，但实际上你只知道他们如何才能最好地按照你的正确想法行事。正如你有权按照自己认为合适的方式生活，他们也一样。与其评判和控制他人，不如集中精力利用你的判断力来制定更好的公正原则。”
  • “独处一段时间。鼓励发展你内向的一面。你会发现，成为一个更全面的人有很多实实在在的好处。”
